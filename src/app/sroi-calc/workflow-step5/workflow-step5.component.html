Deadweight :   It represents the extent to which the outcomes would have happened anyhow.

Displacement :   It occurs when the benefits claimed by a project participant are at the expense of others outside the project.

Attribution :   It represents that some portion of the value created is also contributed by other organisations, initiatives, persons working in the same social bracket.

Drop off :   It considers how a change resulting from a social program will diminish over time.
